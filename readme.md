# qx.ui.list

This package is DEPRECATED and was removed from the main Qooxdoo project and this library only exists for those who have legacy code that depends on it. The recommended library for Data Grids is [qxl.datagrid](https://github.com/qooxdoo/qxl.datagrid). It has a much better design, which is the main reason why `qx.ui.list` was removed from Qooxdoo.
